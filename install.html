Requisitos previos
  • Node.js 18 o superior, o Bun.
  • `npm`, `yarn` o `bun` instalado.
  • Una cuenta de WhatsApp disponible para vincular.
  • API Key de OpenAI, Anthropic, Gemini, Groq u OpenRouter.

Variables de entorno soportadas

VariableDefaultDescripción
PORT3000Puerto de la API HTTP.
AGENT_WS_PORT8081Puerto del servidor WebSocket del agente.
AGENT_WS_HOST127.0.0.1Host donde expone el WebSocket.
PI_PROVIDERopenaiProveedor de IA activo.
PI_MODELgpt-5-miniModelo base usado por el agente.
AGENT_SYSTEM_PROMPT—Prompt del sistema que define el comportamiento global.
HTTP API

`http://localhost:3000`

Agent WebSocket

`ws://localhost:8081`
  1. Abrir `http://localhost:3000`.
  2. Ir a “Administrar Chatbots”.
  3. Ingresar un UUID como `ventas-bot`.
  4. Hacer clic en “Crear chatbot”.
  5. Escanear el QR desde WhatsApp en “Dispositivos vinculados”.

Paso 6: ¡Funcionando! 🎉

Cuando el estado pase a `open`, Chabi restaurará automáticamente la sesión en próximos arranques y tu bot quedará listo para responder.

Solución de problemas

Puerto 8081 ocupado

Cambia `AGENT_WS_PORT` en `.env` y reinicia el proceso.

QR expirado

Espera la regeneración automática o reinicia la sesión para obtener uno nuevo.

El bot no responde

Revisa la API Key configurada, el proveedor activo y los logs del proceso.
